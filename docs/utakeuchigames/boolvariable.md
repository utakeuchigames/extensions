# boolvariable
## 注意！  
・使用するとき、サンドボックスをonにした状態で、インストールをすると、ブロックパレットが壊れるという状態を確認しましたので、まず、javascriptファイル本体をダウンロードして(このマークです<img src="./../../images/utakeuchigames/IMG_0028.jpeg" width="18" style="vertical-align: middle;">)、拡張機能→turbowarp→カスタム拡張機能→ファイルからサンドボックスをoffにした状態でインストールしてください。  
  
## 使い方
インストールした後、ブロックパレットに以下のブロックたちが追加されます。

<img src="./../../images/utakeuchigames/IMG_0060.jpeg" width="300" alt="ブロックメニューの様子">

「真偽値変数」が本アドオンの本体で、機能は以下の通りです。

| ブロック | 説明(ここで出てくる「変数」は本アドオン専用の変数です。scratch純正のブロックからはアクセスできません。) |
| :--- | :--- |
| `変数作成フォームを開く` | 変数を作るフォームを開きます |
| `bool値[(空) ▼]を[true▼ ]にする` | 変数~の値をtrueなどの真偽値にします　|
| `bool値　▼]` | 変数~の値を入手します　|
| `もしbool値[(空) ▼]が[true ▼]になったら` | 変数が決まった値になったら発火します(値が変わった瞬間にしか動きません) |
| `全てのbool値を見る`,`全てのbool値の情報を見る` | 【デバッグ用】全変数の状態を確認できます　|

「その他のキット」は論理演算を補助するツールで、機能は以下の通りです。

| ブロック | 名称 | 説明 |
| :--- | :--- | :--- |
| `!` | ではない | 条件を反転します |
| `&&` | かつ | 両方の条件が真の時に真となります |
| `\|\|` | または | どちらかの条件が真の時に真となります |
| `!==` | 排他的論理和(XOR) | 片方のみが真の時に真となります |

※ `!==` (XOR) はScratch標準ブロックで作成すると複雑になるため、この拡張機能で短縮できるようにしています。
